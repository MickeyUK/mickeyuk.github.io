Coins.js

Javascript

This is a little coin machine example I made using the three.js Javascript 3D library. I used Physijs for the physics. The cabinet and coin models I created in Blender. The sound effects are public domain, sourced from freesound.org. I used a SMAA shader for edge smoothing.

Runs best in Google Chrome, where it can maintain 60fps on modest hardware at 1920x1080.

I will continue to add to this example when I have time, but it hopefully will be a good starting point for anybody wishing to create their own arcade game.
